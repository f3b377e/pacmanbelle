Documentazione delle definizioni

#define DEBUG_CONSOLE

Questo file contiene la debug console, un'interfaccia che viene usata solo in modalità di debug che consente di ricevere in tempo reale le informazioni riguardanti il pacman la mappa e i fantasmi, oltretutto si può scegliere a quale livello/mappa giocare.

Per attivare questa modalità, bisogna compilare il progetto con il "make debug" e una volta in gioco basta premere il tasto "d".
